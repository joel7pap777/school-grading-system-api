# School Grading & Result Management System API

This project is a backend API for managing school grading and student results.

## Features

- User authentication (Admin, Teacher, Student)
- Student management
- Teacher management
- Class and subject management
- Score and grading system
- Result viewing

## Technologies Used

- Python
- Django
- Django REST Framework
- SQLite (development)

## Project Structure

- accounts – authentication and user roles
- students – student profiles
- teachers – teacher profiles
- classes – classes and subjects
- grades – grading and results

## Setup Instructions

1. Clone repository
2. Create virtual environment
3. Install dependencies
